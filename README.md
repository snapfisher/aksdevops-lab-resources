
# Microsoft Technology Center, Reston VA, ADO/AKS Lab 

These are the resources used for the Azure DevOps/AKS lab given at the Microsoft Technology Center in Reston, VA.

Part of this lab, uses Phippy and Friends from [Azure/phippyandfriends](https://github.com/Azure/phippyandfriends).

The actual version used is located here [snapfisher/phippyandfriends](https://github.com/snapfisher/phippyandfriends).  The version in this repository has been modified to work with the lab instructions.
